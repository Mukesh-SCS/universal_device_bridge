# UDB Roadmap

This roadmap outlines the planned features and milestones for Universal Device Bridge (UDB).

---

## v1: Initial Release
- Host-side CLI (adb-style client)
- Linux daemon
- Simulator daemon
- USB and TCP transport support
- Key-based device pairing and authentication
- Core features: command execution, log streaming, file transfer

---

## v2: Expansion & Hardening
- MCU (Microcontroller) support
- Additional transport layers (e.g., serial, BLE, CAN, etc.)
- Protocol hardening and security improvements
- Enhanced error handling and diagnostics

---

## v3: Advanced & Optional Features
- Optional cloud augmentation (never required for local use)
- Fleet management and tooling (opt-in)
- Advanced scripting and automation APIs
- Cross-platform GUI (optional)

---

For more details, see the architecture documentation and follow project updates in the repository.
